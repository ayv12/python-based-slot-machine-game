#  🎰Python-Based-Slot-Machine-Game

A casino-style desktop game built using Python and Tkinter, featuring a graphical user interface, betting mechanics, and randomized slot machine gameplay.

## Features

* Interactive slot machine interface
* Betting system
* Randomized slot outcomes
* Custom game icon
* Casino-themed design
* Simple and user-friendly gameplay

## Technologies Used

* Python
* Tkinter


## Project Overview

This application simulates a slot machine game where users can place bets and spin the reels for randomized results. The project combines GUI development with game logic to create an engaging desktop application.

## How to Run

* Clone the repository
* Navigate to the project folder
* Run:
* python pythonscript.py
